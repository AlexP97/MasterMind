Para compilar el juego, debe introducir:

	$ make

Para jugar al juego una vez compilado, se debe introducir:

	$ make run

Para ejecutar los tests (drivers), debe introducir:

	$ make run_tests

Para ejecutar el test JUnit, debe introducir:

	$ make run_junit
