Carpeta domain

Esta carpeta contiene todos los archivos .java, donde se encuentran todos los drivers, tests y stubs de la Capa de Dominio.

- CodeBreakerDriver.java = este archivo contiene la clase de testeo de CodeBreaker y sus funciones. Se encarga de probar que todas las funciones de CodeBreaker funcionan.

- CodeMakerDriver.java = este archivo contiene la clase de testeo de CodeMaker y sus funciones. Se encarga de probar que todas las funciones de CodeMaker funcionan.

- CodePegDriver.java = este archivo contiene la clase de testeo de CodePegDriver y sus funciones. Se encarga de probar que todas las funciones de CodePegDriver funcionan.

- CodePegStub.java = contiene la clase CodePegStub y sus funciones. Permite crear un CodePeg de forma segura usando tipos primitivos para poder usarla en los drivers de otras 
clases que utilicen funciones de la clase CodePeg.

- GameTest.java = corresponde al JUnit test de la clase game. Permite probar que las funciones de la clase Game funcionan como deben de funcionar, es decir, hacen lo que se 
espera que hagan.

- JugadorDriver.java = este archivo contiene la clase de testeo de JugadorDriver y sus funciones. Se encarga de probar que todas las funciones de JugadorDriver funcionan.

- KeyPegDriver.java = este archivo contiene la clase de testeo de KeyPegDriver y sus funciones. Se encarga de probar que todas las funciones de KeyPegDriver funcionan.

- KeyPegStub.java = contiene la clase KeyPeg y sus funciones. Un KeyPeg corresponde a una ficha de la combinacion que proporciona
el CodeMaker en una jugada. Es una subclase de la clase Casilla.

- RankingDriver.java = este archivo contiene la clase de testeo de RankingDriver y sus funciones. Se encarga de probar que todas las funciones de RankingDriver funcionan.

- Tests.java = corresponde a la clase Tests. Es la clase que permite probar todos los tests en una ejecucion.
